# Getting Started with PromptBio

Welcome! This guide walks you through everything you need to start analyzing biological data in PromptBio.

## 1. Create an Account

Go to [app.promptbio.com](https://app.promptbio.com) and sign up with your institutional or personal email. hey

## 2. Create a Project

A **Project** is the top-level container for your data, analyses, and results.

1. Click **New Project** on the dashboard
2. Give it a name and description
3. Select the organism (Human, Mouse, etc.)
4. Click **Create**

## 3. Upload Your Data

Go to **Data → Upload** and upload your expression matrix and metadata file. See [Data Management](../Data/overview.md) for formatting requirements.

## 4. Run Your First Analysis

Once data is uploaded, go to **Analysis** and choose a module:

- **Differential Analysis** — find significant genes between two groups
- **Comparative Analysis** — compare across multiple conditions
- **Single-cell Clustering** — identify cell populations

## 5. View and Export Results

Results appear in the **Results** panel after each run. You can:
- View plots interactively in the browser
- Download files as CSV or PNG
- Share results with collaborators via a link
